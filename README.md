# 📊 Smart Data Analytics Dashboard

🚀 **Live App:** https://analytics-dashboard-shruti.streamlit.app

---

## 📌 Overview

An interactive **data analytics dashboard** built with Streamlit that allows users to upload datasets and perform **data cleaning, visualization, filtering, and insights generation** without coding.

---

## ✨ Features

* 📂 Upload CSV/Excel files
* 📊 Data summary & overview
* 🧹 Handle missing values (Mean, Median, Mode)
* 📈 Interactive charts (Scatter, Bar, Line, Histogram)
* 🔍 Filtering & Groupby operations
* 🧠 Automated insights & correlation analysis
* ⬇️ Download processed data

---

## 🛠️ Tech Stack

* Streamlit
* Pandas
* Plotly
* Scikit-learn

---

## ⚙️ Run Locally

```bash
git clone https://github.com/shruti-coon/analytics-dashboard.git
cd analytics-dashboard
pip install -r requirements.txt
streamlit run app.py
```
